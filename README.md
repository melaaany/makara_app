# Keuangan Mahasiswa 🎓💸

A human-centered, clean, and intuitive personal finance tracker designed specifically for students. This application helps you manage your daily expenses and income with ease, providing clear insights into your financial health.

## ✨ Features

- **Minimalist Dashboard**: A clean view of your current balance, total income, and total expenses.
- **Human-Centered UI**: Thoughtfully designed with a soft color palette and intuitive interactions to reduce financial stress.
- **Smart Categorization**: Group your spending into student-specific categories like *Jajan*, *Transportasi*, and *Kuliah*.
- **Visual Insights**: Easy-to-understand pie charts showing where your money goes.
- **Transaction History**: A clear, readable list of all your past activities.
- **Responsive Form**: A modern bottom-sheet interface for quick transaction logging.

## 🛠️ Technology Stack

- **Flutter**: For a smooth, cross-platform experience.
- **Provider**: Robust state management ensuring real-time UI updates.
- **FL Chart**: Beautiful and responsive data visualization.
- **Intl**: Proper local currency formatting (Rupiah).

## 🚀 Getting Started

To run this project locally:

1. Clone the repository.
2. Ensure you have the Flutter SDK installed.
3. Run `flutter pub get` to install dependencies.
4. Run `flutter run` to launch the app on your device or emulator.
